# SSUG Status Page

SSUG Status Page 是一个基于 [UptimeFlare](https://github.com/lyc8503/UptimeFlare) 的状态监控页面, 由 VrianCao 汉化, 用于监控 SSUG 各项服务的运行状态。

## 功能特点

- 🚀 轻量级监控页面
- 📊 实时状态展示
- 📈 历史数据统计
- 🔔 状态变更通知
- 🌐 多服务监控
- ⚡ 快速响应

## 监控服务

目前监控的服务包括:

- SSUG 官方网站
- SSUG Matrix 服务器
- Element Web 客户端
- SSO 服务(前端、后端、OAuth)
- 法律数据库
- 导航页面
- AI 服务(Open WebUI)
- Fatpaper 游戏
